Cinema API - Readme

This is the documentation for the Cinema API.

The Cinema API is a RESTful API that allows you to interact with the cinema database.

The Cinema API is built using Node.js, Express.js, and MongoDB.

The Cinema API is hosted on Henry home server.

The Cinema API is open source and available on GitHub.

Movies

The Movies endpoint allows you to interact with the movies in the cinema database.

The Movies endpoint is located at /api/v1/movies

        fetch('http://localhost:3000/api/v1/movies')
            .then(response => response.json())
            .then(data => console.log(data))

Shows

The Shows endpoint allows you to interact with the shows in the cinema database.

The Shows endpoint is located at /api/v1/shows

        fetch('http://localhost:3000/api/v1/shows/:movieId')
        .then(response => response.json())
        .then(data => console.log(data))

Bookings

The Bookings endpoint allows you to interact with the bookings in the cinema database.
